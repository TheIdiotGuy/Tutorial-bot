# Tutorial-bot

> A tutorial bot

## Usage

- Remove .example file extension from `.env.example` & `config.json.example`.
> **.env**
```bash
token = "PUT_TOKEN_HERE"
``` 
> **config.json**
```json
{
    "bot_id": "YOUT_BOT_USER_ID",
    "test_server": "TEST_SERVER_ID"
}
```
